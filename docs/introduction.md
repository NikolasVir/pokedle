# Introduction

Pokedle is a full-stack Pokémon guessing game built with **.NET 10** and
**Angular 21**.

## Stack

| Layer     | Tech                                 |
| --------- | ------------------------------------ |
| API       | .NET 10, HotChocolate (GraphQL)      |
| Database  | PostgreSQL + EF Core                 |
| Frontend  | Angular 21, Tailwind CSS v4          |
| Auth      | JWT (demo: `ash` / `pikachu`)        |
| Real-time | GraphQL subscriptions over WebSocket |

## Key Features

- Daily Pokémon guessing with attribute-based feedback
- Seeded from PokéAPI
- Real-time gameplay via GraphQL subscriptions
